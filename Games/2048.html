2048:

Directions:

Press the green flag to start

Use the arrow keys to match tiles by making tiles with the same number collide.

The goal is to get the 2048 tile

Troubleshooting:

If the arrow keys don't work, click on the game screen.

If the game doesn't load, if on mobile, switch to desktop. If on desktop, make sure that flash is enabled.
